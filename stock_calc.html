公式&說明：

  • 導入範例： Demo
  • 1張 = 1000股; 0.3張 = 300股
  • 持股成本： 買價 x 股數 + 買入手續費
  • 賣出成本： 賣出手續費 + 交易稅
  • 總成本： 買入手續費 + 賣出手續費 + 交易稅
  • 獲利金額： 賣價 x 股數 - 持股成本 - 賣出成本
  • 報酬率： 獲利金額 / 買價 x 股數
  • 根據股價不同的遞增基數自動計算
    • 每股市價未滿10元者，股價升降單位為0.01元
    • 10元至未滿50元者為0.05元
    • 50元至未滿100元者為0.1元
    • 100元至未滿500元者為0.5元
    • 500元至未滿1000元者為1元
    • 1000元以上者為5元
